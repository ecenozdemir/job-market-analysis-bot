# Job Market Gap Analyzer (Kariyer Boşluk Analizi)

## 🌍 Overview (Genel Bakış)
This project is an automated tool designed to track and analyze **ERP Specialist** and **Data Analyst** job postings on LinkedIn. It identifies the gap between market expectations and candidate skills using Python, SQL, and AI.

Bu proje, LinkedIn'deki **ERP Uzmanı** ve **Veri Analisti** iş ilanlarını otomatik olarak tarayan ve piyasa beklentileri ile aday yetenekleri arasındaki boşluğu analiz eden bir araçtır.

## 🚀 Key Features (Temel Özellikler)
* **Automated Scraping:** Daily tracking of job requirements. (Günlük ilan takibi)
* **Skill Gap Analysis:** Comparing required technologies (SQL, Power BI, ERP modules). (Yetenek analizi)
* **Live Dashboard:** Automated updates via GitHub Actions. (GitHub Actions ile canlı güncellemeler)

## 🛠 Tech Stack (Kullanılan Teknolojiler)
* **Language:** Python
* **Database:** SQL (SQLite/PostgreSQL)
* **Automation:** GitHub Actions
